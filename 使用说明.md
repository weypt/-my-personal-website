## ⏰ 定时自动同步（可选）

如果需要每小时自动同步，可以创建定时任务：

1. 按 `Win + R`，输入 `taskschd.msc`，回车
2. 点击 **"创建基本任务"**
3. 名称：`个人网站自动同步`
4. 触发器：选择频率（如每小时）
5. 操作：启动程序
   - 程序：`powershell.exe`
   - 参数：`-ExecutionPolicy Bypass -File "C:\Users\15432\Desktop\my-personal-website\Sync-To-GitHub.ps1"`
6. 完成

## ❓ 常见问题

**Q: 修改后多久能在网站上看到？**  
A: GitHub Pages 通常在 1-2 分钟内完成部署，可通过 https://github.com/weypt/-my-personal-website/actions 查看部署状态

**Q: 同步失败怎么办？**  
A: 检查网络连接，确保能访问 GitHub，然后重新运行同步脚本

**Q: 如何查看同步历史？**  
A: 访问 https://github.com/weypt/-my-personal-website/commits/main
